# TikTokHelper

[English](README.md)

TikTokHelper 在 TikTok 网页上添加了一个下载工具。

它支持下载当前视频或图集作品，支持个人主页中打开的视频弹窗下载，也支持在个人主页中手动选择多个作品后按顺序批量下载。

## 功能

- 下载当前 TikTok 视频。
- 支持下载 TikTok 图集 / 图片类作品。
- 支持 TikTok 推荐页。
- 支持个人主页中打开的视频弹窗。
- 批量下载手动选择的个人主页作品。
- 批量下载按顺序执行。
- 提供下载状态通知。
- 提供设置面板。

## 支持的页面

### 推荐页

在 TikTok 推荐页中，TikTokHelper 会在右侧操作栏添加一个紧凑的下载按钮。

打开按钮菜单后，可以下载当前视频或图集作品。

### 个人页视频弹窗

在 TikTok 个人主页中点击某个视频后，TikTokHelper 会在右上角操作按钮附近添加下载按钮。

该按钮用于下载当前弹窗中打开的视频或图集作品。

### 个人主页批量下载

在 TikTok 个人主页中，TikTokHelper 会在可见作品卡片上显示选择复选框。

使用方式：

1. 打开一个 TikTok 个人主页。
2. 手动勾选想要下载的作品。
3. 点击个人主页顶部操作区附近的 TikTokHelper 批量按钮。
4. 选择 `下载选中`。
5. 在确认弹窗中检查已选作品。
6. 点击 `开始下载`。
