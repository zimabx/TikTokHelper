# TikTokHelper

[简体中文](README.zh-CN.md)

TikTokHelper has added a download tool to the TikTok web pages.

It supports downloading the current video or photo post, downloading videos opened from profile pages, and manually selecting posts on a TikTok profile page for sequential batch downloading.

## Features

- Download the current TikTok video.
- Download TikTok photo/image posts.
- Works on the TikTok recommendation feed.
- Works on profile video pop-up pages.
- Batch download manually selected profile posts.
- Sequential batch downloading.
- Download status notifications.
- Settings panel.

## Supported Pages

### Recommendation Feed

On the TikTok recommendation feed, TikTokHelper adds a compact download button to the action bar.

Use the button menu to download the current video or photo post.

### Profile Video Pop-up

When opening a video from a TikTok profile page, TikTokHelper adds a download button near the top-right action buttons.

This allows downloading the video or photo post currently opened in the profile pop-up.

### Profile Page Batch Download

On a TikTok profile page, TikTokHelper shows selection checkboxes on visible post cards.

Usage:

1. Open a TikTok profile page.
2. Manually select the posts you want to download.
3. Click the TikTokHelper batch button near the profile action buttons.
4. Choose `Download selected`.
5. Review the selected posts in the confirmation dialog.
6. Click `Start download`.

## Important Notes

- Only download content that you have the right or permission to save.
- This script does not bypass paid content, private content, account restrictions, or DRM.
- TikTok frequently changes its web page structure, so some features may break after TikTok updates.
- If something stops working, please report the page type and describe the issue.
